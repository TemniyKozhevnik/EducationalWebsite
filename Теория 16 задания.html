Теория для 16 задания КЕГЭ по информатике

Наиболее часто используемые функции
КомандаДействиеСинтаксис
СМЕЩЭта функция позволяет возвращать ссылку на диапазон, который смещен относительно исходной ссылки на определенное число столбцов и строк.СМЕЩ(ссылка; смещение по строкам; смещение по столбцам; высота; ширина)
СУММСуммирует все числа в интервале ячеекСУММ(число1;число2)
СЧЁТПодсчитывает количество всех непустых значений указанных ячеекСЧЁТ(значение1, [значение2],…)
СРЗНАЧВозвращает среднее значение всех непустых значений указанных ячеекСРЕДНЕЕ(число1, [число2],…)
МАКСВозвращает наибольшее значение из набора значений МАКС(число1;число2; …)
МИНВозвращает наименьшее значение из набора значенийМИН(число1;число2; …)
ЕСЛИ Проверка условия. Функция с тремя аргументами: первый аргумент — логическое выражение; если значение первого аргумента — истина, то результатом выполнения функции является второй аргумент. Если ложно — третий аргумент.ЕСЛИ(лог_выражение;значение_если_истина;значение_если_ложь)
СЧЁТЕСЛИ Количество непустых ячеек в указанном диапазоне, удовлетворяющих заданному условию.СЧЁТЕСЛИ(диапазон, критерий)
СУММЕСЛИСумма непустых ячеек в указанном диапазоне, удовлетворяющих заданному условию.СУММЕСЛИ(диапазон, критерий, [диапазон_суммирования])
